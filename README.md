# CV Interactivo — María Martín Vélez

Portfolio personal desarrollado con React y Vite. Presenta mi trayectoria 
en formato changelog versionado, con animaciones al hacer scroll basadas 
en IntersectionObserver nativo (sin librerías externas).

## Stack

- React 19 + Vite 8
- CSS vanilla con variables
- IntersectionObserver API para las animaciones
- Desplegado en GitHub Pages
